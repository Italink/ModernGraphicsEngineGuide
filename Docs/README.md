# Modern Graphics Engine Guide

该系列文章旨在归纳现代图形引擎开发必备的基本技术路线，它并不是一个细致入微的教程，也没有包含太多的高级篇章，这些文章或许能让初学者少走弯路 ，把更多的精力放在更高级的理论技术上面，在此之前，读者至少要达到一个这样的境界  — **I can do anything if I want**

文章的内容将包含：

- **现代C++开发工作流** 

- **GUI及工程架构**

- **现代化的图形API** 

- **Unreal Engine 5**
